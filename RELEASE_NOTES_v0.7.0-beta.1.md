# Release Notes — v0.7.0-beta.1

This is the first official beta release of Jellyfin Invite Theme.

## What works

- Creating invites
- Viewing invite links
- Public signup form
- Creating Jellyfin users
- Logging registrations
- Tracking who an invite was assigned to
- Deleting invites from admin

## Known limitations

- Older invites created before plain-token storage may not show a full invite link
- This is still a beta release and should be tested before production use

- Add screenshots to the GitHub repo

- Prepare `v0.7.0-beta.2` after feedback
